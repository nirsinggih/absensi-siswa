# absensi-siswa
Aplikasi absensi siswa menggunakan cr-code, rf-id yang ringan sesuai format baku administrasi sekolah

Dengan adanya AI membuat aplikasi menjadi semakin mudah.

Aplikasi ini fokus untuk mempermudah rutinitas absensi siswa dan sesuai format baku administrasi sekolah.

Dibagikan secara gratis, semoga berkah.
